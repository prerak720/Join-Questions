Overview

This repository contains two sample datasets designed for practicing SQL joins, aggregations, subqueries, and advanced logical queries commonly asked in Data Analyst interviews.

The dataset includes:

Orders Table

Customers Table

These tables help you practice INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN, CROSS JOIN, and complex SQL logic such as HAVING, grouping, window functions, and subqueries.
